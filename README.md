# ArchivesSpace plugin for customizing frontend and public favicons

This plugin is intended to work with ArchivesSpace v4.0.0.

```
.
├── README.md
├── frontend
│   ├── assets
│   │   ├── favicon.png
│   │   └── favicon.svg
│   └── views
│       └── site
│           └── _favicon.html.erb
└── public
    ├── assets
    │   ├── favicon.png
    │   └── favicon.svg
    └── views
        └── shared
            └── _favicon.html.erb
```
